# ssm_RESTful
一个前后端分离的简易ssm框架实例

数据库users表结构：
+-------+-----------+------+-----+---------+----------------+ 
| Field | Type      | Null | Key | Default | Extra          | 
+-------+-----------+------+-----+---------+----------------+ 
| id    | int(4)    | NO   | PRI | NULL    | auto_increment | 
| name  | char(255) | YES  |     | NULL    |                | 
| sex   | int(4)    | NO   |     | 0       |                | 
| info  | char(255) | YES  |     | NULL    |                | 
| pwd   | char(25)  | YES  |     | NULL    |                | 
+-------+-----------+------+-----+---------+----------------+
